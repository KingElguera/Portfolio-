# Photo de profil

Pour que ta photo s’affiche sur le portfolio, mets **une seule** image dans ce dossier **assets**, avec **un** de ces noms :

- **photo.jpg**  
- **photo.png**

Le site essaie dans cet ordre : `photo.jpg` → `photo.png` → puis un autre nom si tu l’as déjà utilisé avant.

**À faire :**
1. Ouvre le dossier **Portfolio** sur ton Bureau.
2. Ouvre le sous-dossier **assets**.
3. Glisse-dépose ta photo dedans.
4. **Renomme le fichier** en **photo.jpg** (ou **photo.png** si c’est un PNG).
   - Clic droit sur le fichier → « Renommer » (ou sélectionne et appuie sur Entrée).
   - Remplace tout le nom par exactement : `photo.jpg` ou `photo.jpg`.

Si tu ne mets pas de fichier ou que le nom est différent, un placeholder « Photo » s’affichera à la place.

**Conseil :** image carrée ou portrait (ex. 400×400 px), format JPG ou PNG.
